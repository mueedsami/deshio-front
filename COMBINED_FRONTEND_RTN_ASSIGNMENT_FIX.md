# Combined Deshio Frontend Fix

This package includes both frontend panels:

1. Dispatch RTN Repair
   - `/inventory/dispatch-rtn-repair`
   - product-wise details, per-product dry run, per-product apply, global confirmation flow

2. Assignment Blockers
   - `/inventory/assignment-blockers`
   - return-aware blocker details, ordered quantity, returned quantity, effective hold quantity, per-row release action

3. Dispatch page pagination
   - `/inventory/outlet-stock`

This build is based on the return-aware frontend package, which already contained the Dispatch RTN Repair page and pagination changes.
